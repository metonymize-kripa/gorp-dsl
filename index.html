How It Works: The 3-Layer Flow

1

Expert Defines the "What" with DSL

A domain expert writes a simple spec: what data to use, what capabilities to apply (e.g., `forecast`), and what UI elements to show (e.g., `chart`, `slider`).

‚Üì
2

Hosted Renderer Does the "How"

Our cloud runtime interprets the DSL, securely connects to data sources, executes the requested capabilities from the open-source kit, and prepares the results.

3

UI is Auto-Generated for the User

The end-user gets a clean, interactive web interface‚Äîwith the charts, sliders, and buttons defined in the DSL‚Äîwithout any front-end coding required.
